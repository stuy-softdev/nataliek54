Basics

Bootstrap is a frontend tool for HTML, JavaScript, and CSS. It handles CSS templates that help with responsive pages.

Starting Out

Bootstrap is a CSS, HTML, and JS library, meaning that it's full of templates that you can access. You can connect bootstrap to your code by pasting in the link into the head of your HTML like so: 


      <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-sRIl4kxILFvY47J16cr9ZwB07vP4J8+LH7qKQnuqkuIAvNWLzeN8tE5YBujZqJLB" crossorigin="anonymous">

and this link into the body: 


      <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/js/bootstrap.bundle.min.js" integrity="sha384-FKyoEForCGlyvwx9Hj09JcYn3nv7wiPVlz7YYwJrWVcXK/BmnVDxM+D2scQbITxI" crossorigin="anonymous">>

Then, for each element you want, you can go to the Bootstrap Page and search for the code for whatever element you want. Bootstrap will give an example of the class of whatever you're looking for (ie 'carousel' or 'grid') and you can paste that into your code.
